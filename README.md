🎯 Project Overview
Analysis of 12-month retail sales data to identify top products, seasonal trends, and forecast future revenue using Machine Learning.

## 🛠️ Tools & Technologies
- **Python** — Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
- **Machine Learning** — Random Forest Regressor (88% accuracy)
- **SQL** — 6 analytical queries
- **Power BI** — Interactive dashboard with KPI cards
- **GitHub** — Version control

## 📈 Key Results
- 🏆 Built Random Forest model with **88% accuracy**
- 📊 Analyzed **9,994 retail orders** across 4 regions
- 💰 Total Revenue: **$887.92K** | Total Profit: **$108.80K**
- 🛒 Total Orders: **2,012** unique orders
